Vector class

Constructor

  • Constructor 1:

    • Vector(comlex_str: str) -> Vector by complex type from input.


  • Constructor 2:

    • Vector(position: Iterable) -> Vector with two element of iterable.

    Caution: The param 'position' can be another vector.


  • Constructor 3:

    • Vector(start_point, end_point) -> Vector by linking two points.

    Caution: Two points must be typed Vector.

  • Constructor 4:

    • Vector(x, y) -> Vector with position is x, y.

  • Constructor 5:

    • Vector() -> Vector with position is 0, 0.



Static Method

  • length_of()

    • Vector.length_of(vector) -> float is the lenght of vector folllow the fomula sqrt(x**2 + y**2).

  • get_item_from_complex()

    • Vector.get_item_from_complex(complex_str) -> Vector with complex (use c for angle complex and i for normal complex), for more details, run print(pphysics.Vector.get_item_from_complex.__doc__) .

Class Method

  • analyze()

    • analyze(i_unit_vector, j_unit_vector) -> tuple[_Float], two float number m,n follow fomula vector = m * i + n * j

  • stem_parallelogram_rule()

    • stem_parallelogram_rule(i_same_direction_vector, j_same_direction_vector) -> tuple[_Vector], two vector u, v follow fomula vector = u + v (with u and i is the same direction, v and j is the same direction).

Properties

  • length

    • Return the length of self, same the method Vector.length_of(self).

  • position

    • Return a tuple of x and y.
    Can set this property.

  • x

    • Return value of x.
    Can set this property.

  • y

    • Return value of y.
    Can set this property.

Math Support

  • Addition

    • Can do the addition between two vector.
    Ex:Vector(1, 2) + Vector(2, 4) = Vector(3, 6).
    With subtraction similar: Vector(3, 4) - Vector(3, 2) = Vector(0, 2)

  • Multipication

    • Can do the multipication between two vector or a vector with a number.
    Ex: Vector(1, 2) * 3 = Vector(2, 4).
    Two vector: Vector (2, 3) * Vector(4, 5) = 23.
    With divition to a number similar: Vector(10, 5) / 5 = Vector(2, 1).

  • Power

    • Can pow the vector to a number.
    Ex: Vector(1, 2) ** 2 = 5. Follow the step, if pow index is odd, the return is Vector, is number if even.

  • Absolution

    • Call abs on self will return the length of Vector.

  • Iteration

    • Can access to a Vector, get item, set item or run in for loop like a tuple.

  • toString

    • Return the string of tuple of x, y.
